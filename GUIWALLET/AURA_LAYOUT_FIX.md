# AURA Layout Fix

Changes:
- AURA no longer floats over the header action buttons.
- Top-right actions are grouped in one row:
  - Chat with AURA AI
  - Start node
  - Refresh
  - Stop node
- AURA can now be expanded with `Large view`.
- Responsive behavior improved for smaller MacBook screens.

Note: If the app opens directly on Wallets instead of the welcome/initial screen, that is expected when a previous wallet/session exists in local app data.
