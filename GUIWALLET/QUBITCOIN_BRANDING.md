# QUBITCOIN Branding

The QRX network currency is **QUBITCOIN** with ticker **QUB**.

Use in UI:
- Currency name: QUBITCOIN
- Ticker: QUB
- Examples: `0.00 QUB`, `Send QUB`, `Receive QUB`
- Technical project/network references may still use QRX where they refer to the chain/core/project name.

Do not rename technical binaries unless the core project itself changes:
- `qrx`
- `qrxd`
- `qrx-cli`
