First-start safety briefing

Unique selling point

QRX is built as a quantum-ready, lightweight chain with local-first wallet control.

Your keys stay local. The wallet can run as a safe home wallet, a passive delegator, or an advanced validator node. Core consensus stays clean, transparent and independent.

Local-firstWallet keys and recovery stay on your machine.
Delegation-readyEarn through validators without running infrastructure.
Validator-gradeAdvanced users can run a daemon and secure the network.
Operating modes

Choose the right mode: Wallet, Delegator, or Validator.

Wallet Mode

Send, receive and keep a local wallet. You may run the daemon without producing blocks. No slashing risk.

Delegator Mode

Delegate QUB to a validator. Your computer can be offline. Validator performance affects rewards, but you do not run infrastructure.

Validator Mode

Run a block-producing node with self-stake. This requires stable 24/7 uptime and carries slashing risk.

Best practice

Home computers are best for Wallet/Delegator Mode. Validator Mode belongs on stable servers.

Recommended setup:

Use Wallet Mode by default. Use Delegator Mode for passive participation. Enable Validator Mode only on a reliable VPS/server, with monitoring, backups, and stable network availability.

Staking

Delegator Mode is safe for home users. You can delegate QUB and close the wallet. Slashing/offline penalties apply to validators, not to normal wallet users. Self-staking makes this wallet a validator candidate and requires Validator Mode.
Validator risk:

Only self-stake if Validator Mode is enabled and this machine/server is intended to run reliably. Minimum self-stake is 100 QUB.

BTC on the Rocks

Put BTC into a controlled swap flow without giving custody to the app. Your keys remain local.

QUBITCOIN can use SHA3 internally, but BTC atomic swaps require SHA256 hashlocks. Quantum Swaps v1 therefore uses SHA256 for BTC ↔ QUB compatibility.

Swap safety model

1. QuoteRate, fees and timeout shown
2. LockFunds are time-locked
3. RedeemSecret unlocks counterparty tx
4. RefundTimeout enables recovery
